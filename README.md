## Overview

A comprehensive, web-based School Management Information System designed specifically for Ghanaian Senior High Schools. This system automates student registration, academic assessments, grading, and reporting in compliance with Ghana Education Service (GES) standards.

## Features

### Core Modules
- **Student Registration & Management**
- **Academic Assessment System** (9 subjects per student)
- **Automatic Grading & Ranking** (A1-F9 grading system)
- **Class Average Calculations**
- **Promotion Decision Logic**
- **Headmaster's Terminal Dashboard**
- **GES Printable Report Cards**

### Analytics & Reporting
- **Power BI Dashboard Integration**
- **Real-time Performance Analytics**
- **Programme-wise Comparison**
- **Attendance Monitoring**
- **Promotion Rate Analysis**

### Output Formats
- **PDF Report Cards** (GES Official Format)
- **Excel Exports**
- **CSV Data Dumps**
- **Printable Registers**
